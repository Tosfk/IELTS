# 1.Line Graph

## Essay Structure
1. Introduction
2. Overview (也可以放到最后)
3. Detail
4. Detial

### 1 Introduction
一个句子，描述问题，将题干同义替换

### 2 Overview
2句，The main, general thing  
找到两个要点

### 3+4 Details
有数字则比较，有趋势则描述。  
2段是为了更好的organize和group。
No conclusion!
